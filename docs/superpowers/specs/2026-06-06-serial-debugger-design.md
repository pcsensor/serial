# 串口调试助手设计规格

## 概述

基于 Rust + Dioxus + Tauri v2 构建的跨平台（Windows / macOS / Linux）串口调试助手，支持多编码收发、预设指令管理、数据导出，采用现代化毛玻璃透明 UI。

## 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | Tauri v2 | 窗口管理、IPC、系统集成 |
| 前端 | Dioxus (WASM) | 组件化 UI |
| 串口通信 | serialport crate | 跨平台串口操作 |
| 毛玻璃 | window-vibrancy crate | 系统级 vibrancy/Mica |
| 持久化 | tauri-plugin-store | JSON 存储 |
| 文件对话框 | tauri-plugin-dialog | 系统原生对话框 |

## 串口通信模块

### 核心结构

```rust
struct SerialManager {
    port: Option<Box<dyn SerialPort>>,
    receiver_thread: Option<JoinHandle<()>>,
    event_emitter: EventEmitter,
}
```

### Tauri Commands

| Command | 功能 |
|---------|------|
| `list_ports` | 枚举可用串口，返回端口列表 |
| `open_port` | 打开指定串口（含参数配置） |
| `close_port` | 关闭当前串口 |
| `send_data` | 发送数据（支持 ASCII/HEX/自定义编码） |
| `get_port_config` | 获取当前串口配置 |

### 串口参数

- 端口：从 `list_ports` 动态获取
- 波特率：常用值（9600/19200/38400/57600/115200 等）
- 数据位：5/6/7/8
- 停止位：1/1.5/2
- 校验位：None/Odd/Even/Mark/Space
- 流控：None/RTS-CTS/XON-XOFF

### 数据接收流程

```
串口数据到达 → 接收线程读取 → 解码（ASCII/HEX/编码）
    → 添加时间戳 → Tauri Event("serial-data") → 前端接收显示
```

- 接收线程独立运行，不阻塞 UI
- 通过 Tauri Event 推送，前端用 `listen` 监听
- 关闭串口时自动终止接收线程

### 错误处理

- 串口断开自动检测并通知前端
- 所有 Command 返回 `Result<T, String>`，前端统一展示错误

### 现代化效果

| 效果 | 实现方式 |
|------|----------|
| 窗口透明 | Tauri `transparent: true` |
| 系统毛玻璃 | `window-vibrancy` crate（macOS vibrancy / Windows Mica） |
| 渐变背景 | CSS `linear-gradient` + `rgba` |
| 卡片毛玻璃 | CSS `backdrop-filter: blur(20px)` + 半透明背景 |
| 圆角阴影 | CSS `border-radius` + `box-shadow` |
| 自定义标题栏 | Tauri `decorations: false` + `data-tauri-drag-region` |

### 状态管理

- **Dioxus Signals**：UI 状态（当前标签页、串口配置、接收数据列表等）
- **Tauri Store Plugin**：持久化存储预设指令（JSON）
- **全局状态**：串口连接状态、收发字节统计

## 数据格式支持

| 格式 | 说明 |
|------|------|
| ASCII | 纯文本 |
| HEX | 十六进制，如 `48 65 6C 6C 6F` |
| GBK | 中文编码 |
| UTF-8 | Unicode 编码 |

## 数据导出

### 文本文件（.txt）

```
[2026-06-06 12:30:45] 收到: 48 65 6C 6C 6F
[2026-06-06 12:30:46] 发送: Hello
```

### CSV 文件（.csv）

```csv
时间,方向,编码,数据
2026-06-06 12:30:45,接收,HEX,48 65 6C 6C 6F
2026-06-06 12:30:46,发送,ASCII,Hello
```

- 导出通过 Tauri `dialog` 插件弹出系统文件保存对话框
- 导出操作在 Rust 后端执行

## 持久化存储

| 数据 | 存储方式 | 格式 |
|------|----------|------|
| 预设指令 | Tauri Store Plugin | JSON |
| 上次串口配置 | Tauri Store Plugin | JSON |
| 窗口大小/位置 | Tauri 内置 | 自动记忆 |
