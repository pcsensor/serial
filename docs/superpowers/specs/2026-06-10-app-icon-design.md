# 串口调试助手 — 应用图标设计规格

## 概述

为「串口调试助手」设计并生成全套应用图标，替换当前 Tauri 默认图标。

## 设计方案

**方案**：DB9 串口接头正面特写（方案 A）

### 视觉构成

| 元素 | 描述 |
|------|------|
| 底板 | 深色渐变（#1a1a2e → #16213e）圆角矩形，带内阴影 |
| DB9 外壳 | 正面视角梯形金属外壳，银灰渐变（#b8c6db → #8a9bb5），两侧螺丝孔 |
| 针脚 | 上排 5 个 + 下排 4 个，圆形金属针脚 |
| LED 发光 | 2-3 个针脚带绿色辉光（#00ff88），表示数据传输 |
| 3D 效果 | 通过 SVG 渐变、阴影（drop-shadow）、高光实现 |

### 配色

- 背景：深蓝黑 #1a1a2e / #16213e
- 金属：银灰 #b8c6db / #8a9bb5
- 信号绿：#00ff88（辉光效果）
- 暗部：#0d1117

## 实现方式

1. 使用 SVG 手工绘制图标源文件（1024x1024）
2. 通过 `tauri icon` CLI 命令从源文件自动生成全套目标图标
3. 替换 `src-tauri/icons/` 下所有文件

## 输出产物

替换以下文件（保持目录结构不变）：

- `icons/icon.png`（512x512）
- `icons/icon.icns`（macOS）
- `icons/icon.ico`（Windows）
- `icons/32x32.png`、`icons/128x128.png`、`icons/128x128@2x.png`
- `icons/64x64.png`
- `icons/Square*.png`（Windows 磁贴）
- `icons/StoreLogo.png`
- `icons/android/`、`icons/ios/`（移动端）

## 约束

- 小尺寸（32x32）下接头轮廓仍需可辨识
- 不使用外部版权素材，纯代码生成
- SVG 源文件保留在项目中以便后续修改
